# How to Run This Project

You can run this project using one of the following methods:

## Method 1: Using VS Code Live Server (Recommended)
1. Install the **Live Server** extension in VS Code.
2. Open `index.html`.
3. Click "Go Live" at the bottom right corner of the window.

## Method 2: Using npx (Node.js)
If you have Node.js installed, run the following command in your terminal:
```bash
npx live-server
```

## Method 3: Opening Directly
You can simply double-click the `index.html` file to open it in your browser, although some features might require a local server to work correctly.
